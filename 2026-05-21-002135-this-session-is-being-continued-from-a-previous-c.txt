  1. https://github.com/new 접속

  3. Public 또는 Private 선택 → Create repository 클릭
  4. 생성된 페이지에서 …or push an existing repository 섹션의 명령어 복사해서
  터미널에 붙여넣기:

  git remote add origin https://github.com/[내아이디]/life-scenario-planner.git
  git branch -M main
  git push -u origin main

  ---
  STEP 3. Streamlit Cloud 배포

  1. https://share.streamlit.io 접속
  2. Sign in with GitHub 클릭 → 내 GitHub 계정으로 로그인
  3. 우측 상단 "New app" 버튼 클릭
  4. 아래처럼 입력:

  ┌────────────────┬──────────────────────────────────┐
  │      항목      │                값                │
  ├────────────────┼──────────────────────────────────┤
  │ Repository     │ [내아이디]/life-scenario-planner │
  ├────────────────┼──────────────────────────────────┤
  │ Branch         │ main                             │
  ├────────────────┼──────────────────────────────────┤
  │ Main file path │ app.py                           │
  └────────────────┴──────────────────────────────────┘

  GEMINI_API_KEY = "AIzaSyC34Tv-L0UOJ3IteG3VtgNwxKAN4EVZtvo"

  ▎ 배포 후에도 앱 대시보드 → Settings → Secrets 에서 언제든 수정 가능합니다.

  4. "Deploy!" 클릭
