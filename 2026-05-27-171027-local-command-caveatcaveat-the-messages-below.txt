  ---
  Google OAuth 2.0 구현 완료. 다음 설정이 필요합니다:

  .env 파일에 추가

  GOOGLE_CLIENT_ID=발급받은_클라이언트_ID
  GOOGLE_CLIENT_SECRET=발급받은_클라이언트_시크릿
  REDIRECT_URI=http://localhost:8501

  Google Cloud Console 설정 절차

  1. https://console.cloud.google.com → APIs & Services → Credentials
  2. "+ CREATE CREDENTIALS" → OAuth client ID
  3. Application type: Web application
  4. Authorized redirect URIs에 http://localhost:8501 추가
  5. 생성 후 Client ID, Client Secret 복사 → .env에 붙여넣기
  6. OAuth consent screen → 앱 이름, 이메일 설정 (테스트 모드에서 본인 이메일
  추가)

  동작 방식

  - 미로그인 시: 로그인 버튼만 표시 → st.stop()
  - 로그인 후: 사이드바에 프로필 사진 + 이름 + 이메일 + 로그아웃 버튼
  - 로그아웃 클릭 시: 세션 클리어 → 로그인 페이지로 복귀
